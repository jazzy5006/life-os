# 🌸 Jasmine’s Life OS (v1.0)

Welcome to your digital command center. This **Life OS** is a custom-built, responsive web dashboard designed to bridge the gap between scientific precision and a minimalist kawaii aesthetic. It’s built to manage the delicate balance of **clinical trial work**, **puppy parenting (Timber!)**, and **personal health protocols**.

## ✨ Core Features

### 📅 The Master Flow

A dynamic timeline that tracks your day from the 8:30 AM fasting break to the 11:30 PM sleep prep.

* **Contextual Filtering:** Toggle between **Work**, **Puppy**, and **Medical** views to focus on what matters most in the moment.
* **"Now" Indicator:** A real-time pulse that highlights your current task and lets you "Jump to Now" with a single tap.
* **Dr. Yin Protocol Integration:** Baked-in logic for Timber’s socialization and potty breaks.

### 💪 PT Engine

A high-performance exercise tracker that turns physical therapy into a "movement snack."

* **Smart Scheduling:** Automatically switches exercise groups based on the day of the week (e.g., Floor exercises on M/W/F, Standing on T/Th/Sat).
* **Persistence:** Uses `localStorage` to save your progress, so your 0–100% completion bar stays accurate even if you refresh the page.
* **Pomodoro Ready:** Designed to be used during 5-minute work breaks.

### 🌙 Adaptive Aesthetic & Night Mode

* **Kawaii UI:** A pastel-heavy palette featuring "Sophia Yin" lavender, "Timber" mint, and "Medical" peach.
* **Automatic Night Mode:** The interface automatically shifts to a deep, low-eye-strain palette between **8:00 PM and 6:00 AM** to protect your circadian rhythm.
* **Fully Responsive:** A tailored experience for both the mobile "Flow" view and a data-rich Desktop layout.

## 🚀 How to Use

1. **Launch:** Open the `.html` file in any modern browser (Chrome or Safari recommended for mobile).
2. **Reset Today:** Use the **☀️ Reset** button every morning to clear your PT checkboxes and start fresh.
3. **Sync:** Because this uses `localStorage`, your progress is saved to the specific device you are using.
> **Note:** For the best experience, save the file as a "Web App" to your home screen to hide the browser UI.
